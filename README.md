## Project Description
A Python-based Contact Management System that allows users to store, search, update, delete, and manage contacts using dictionaries and functions. The system supports file handling, validation, and data persistence using JSON.

## What I Learned
- Functions and modular programming
- Dictionaries and nested data structures
- String methods for input cleaning
- File operations using JSON
- Input validation and error handling
- Basic testing using assertions

## Features
- Add new contacts with validation
- Search contacts using partial name matching
- Update and delete contacts
- View all stored contacts
- Export contacts to CSV
- View contact statistics
- Persistent storage using JSON file
